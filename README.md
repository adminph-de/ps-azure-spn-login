## Usage

```json
change the JSON file (spn-login.json)

[
    {
       "object": {
           "TENANT_ID": "0000000-0000-0000-0000-00000000000",
           "SPN_ID": "0000000-0000-0000-0000-00000000000",
           "SPN_PW": "yourSecretSPNpassword"
     }
    }
]
```
